Teema:
(Näiteks) Prügi sorteerimine
1. leht: Märgi, millised prügikonteinerid sul kodus on
2. leht: Loetelu mis käib kuhu, otsimisvõimalus olemas. Juhul kui valisid nt olmeprügi ja biojäätmed, siis filtreerib ainult biojäätmed ja ülejäänud on olmeprügi. Juhul kui valisid nt olme, bio ja papp- ja paberpakend, siis filtreerib bio ja pakendid eraldi, ülejäänud olme JNE.
3. leht: Nt "Aitäh, et sorteerid" + mingid faktid prügi kohta vms
